Open Marshmallow Climber.exe to play the game.

Gameplay footage: https://youtu.be/w4oWrNBYtzE 

![](https://raw.githubusercontent.com/c-pj/marshmallow-climber/refs/heads/main/screenshot.png)  

Controls:  
    &emsp;- W = Jump (Press while in air to double jump.)  
    &emsp;- A = Move Left  
    &emsp;- D = Move Right  
    &emsp;- Enter = Restart
